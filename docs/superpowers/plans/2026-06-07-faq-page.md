# FAQ Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a public `/faq` route with an accordion-style FAQ page covering Getting Started, BYOK & API Keys, Security & Privacy, and Pricing.

**Architecture:** A single `FaqPage` component holds all FAQ content as a typed data array. Each category renders a list of `AccordionItem` sub-components that toggle open/closed via local React state. The footer's "Docs & FAQs" link is wired to `/faq` via React Router.

**Tech Stack:** React 18, TypeScript, CSS Modules, React Router v6, `usePageTitle` hook (existing at `src/lib/usePageTitle.ts`)

function AccordionItem({ question, answer }: FaqItem) {
  const [open, setOpen] = useState(false)

  return (
    <div className={[styles.item, open ? styles.itemOpen : ''].filter(Boolean).join(' ')}>
      <button className={styles.question} onClick={() => setOpen((o) => !o)} aria-expanded={open}>
        <span>{question}</span>
        <svg
          className={styles.chevron}
          viewBox="0 0 16 16"
          fill="none"
          stroke="currentColor"

          strokeLinecap="round"
          strokeLinejoin="round"
          aria-hidden="true"

        </svg>
      </button>
      <div className={styles.answer} style={open ? { maxHeight: '600px' } : undefined}>

      </div>

  )
}
